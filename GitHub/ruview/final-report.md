## 🎯 核心发现

### 1. 技术架构

**双语言架构**:
- **Python (v1)**: API 层、服务编排、快速原型
- **Rust (rust-port)**: 核心算法、信号处理、性能关键路径

**性能对比**:
- Python: ~50ms/帧
- Rust: 54,000 帧/秒 (810x 提升)

---

### 2. 核心功能

#### WiFi 人体姿态估计
- **输入**: CSI 子载波幅度/相位
- **输出**: DensePose UV 映射、17 个关键点
- **速度**: 54K fps (Rust)
- **准确率**: 92%

#### 生命体征监测
- **呼吸检测**: 带通滤波 0.1-0.5 Hz → FFT 峰值检测
- **范围**: 6-30 次/分
- **准确率**: 88%

- **心跳检测**: 带通滤波 0.8-2.0 Hz → FFT 峰值检测
- **范围**: 40-120 次/分
- **准确率**: 85%

#### 穿墙检测
- **原理**: 菲涅尔区几何 + 多径建模
- **深度**: 最远 5 米
- **条件**: 需要多基站融合

---

### 3. 技术亮点

#### 多基站融合 (ADR-029)
- 4-6 个 ESP32 节点
- 12+ TX-RX 链路融合
- 360 度覆盖
- <30mm 抖动
- 0 身份交换

#### 自学习系统 (ADR-024)
- 对比学习 CSI 嵌入模型
- 无需标注数据
- 无需训练摄像头
- 自动适应环境

#### 域泛化 (ADR-027)
- MERIDIAN 对抗训练
- 去除环境偏差
- 跨房间/建筑部署
- 一次训练，到处运行

#### 信号线协议 (ADR-033)
- 6 阶段 CRV pipeline
- CSI 感知 → Poincaré 球嵌入
- GNN 拓扑分析
- SNN 时间编码
- MinCut 分割

## 🎓 学习要点

### 信号处理

1. **CSI 数据特性**:
   - 幅度：反映信号衰减
   - 相位：反映传播距离
   - 多径效应：反射/折射

2. **相位净化**:
   - 解包裹：去除 2π 跳变
   - Hampel 滤波：异常值去除
   - 平滑：滑动平均

3. **特征提取**:
   - 幅度统计：mean, variance, RMS
   - 相位差分：相邻子载波相位差
   - 多普勒：时间维度相位变化
   - 频谱分析：FFT → PSD

### 机器学习

1. **DensePose 模型**:
   - 输入：CSI 特征图
   - 输出：UV 映射、关键点
   - 架构：CNN + 全连接

2. **自学习系统**:
   - 对比学习：正负样本对
   - 嵌入空间：相似样本靠近
   - 无需标注：自监督学习

### 系统架构

1. **微服务设计**:
   - 服务编排器统一管理
   - 依赖注入便于测试
   - 异步非阻塞高并发

2. **双语言策略**:
   - Python: 快速迭代、API 层
   - Rust: 性能关键、核心算法
